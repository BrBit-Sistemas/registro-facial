## 🚀 **Como Usar Esta Release**

### 🛠️ **Desenvolvimento**
```bash
# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build com Turbopack
npm run build:turbo

# Linting
npm run lint
```

### 🐳 **Docker**
```bash
# Build da imagem
docker build -t sagep-facial-cpma .

# Executar container
docker run -p 3000:3000 sagep-facial-cpma

# Docker Compose
docker-compose up -d
```
